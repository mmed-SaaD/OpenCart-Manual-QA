# OpenCart Manual QA Project

### Overview

The main objectif of this project is to review all critical user workflows and ensure their consistency, the app's usability and reliability.

### SUT

- Application : **OpenCart 3.0.5.0**
- Environment : **Local Docker deployment**
- Device : **Laptop Only**
- Browser : **Microsoft Edge (latest)**
- Testing Type : **Manual**
- In Scope : 
    * Functional testing
    * Exploratory testing
    * UI/UX testing
    * Usability testing
    * Basic accessibility observations
    * Bug reporting

### Deliverables

The following documents will be delivered by the end of this project :

- Test Strategy
- Test Plan
- Requirements Analysis
- RTM
- Test Scenarios
- Test Cases
- Exploratory Testing Notes
- Bug Reports
- Final QA Report
